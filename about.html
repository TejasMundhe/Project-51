About This Project


This project was made to explore the basics of html through a fun and interesting website. I did this through origami in which this website presents different objects that beginners can make and their details. Enjoy creating something new with origami or just have fun exploring the website.

Goals


  • Learn about making websites using html and css

  • Continue taking steps into the world of computer science
